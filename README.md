# Лабораторная работа: Система часов с устройствами

## Описание

В данной лабораторной работе реализована система управления различными устройствами времени на языке Java с использованием JavaFX для визуализации.  
Система состоит из центрального таймера (`CentralTimeTicker`), который управляет временем и уведомляет подключенные устройства (`TimeObserver`).  
Поддерживаются три типа устройств с разными механизмами работы:

1. **Секундомер (StopwatchDevice)** – отслеживает прошедшее время с момента старта.
2. **Таймер (TimerDevice)** – считает оставшееся время до нуля и уведомляет о завершении.
3. **Будильник (AlarmClockDevice)** – срабатывает в заданное время и поддерживает включение/выключение.

Каждое устройство реализует интерфейс `DeviceController`, что позволяет использовать единый способ управления (старт, стоп, отображение состояния).

## Проблемы и решения (паттерн Observer)

| Проблема | Паттерн | Решение проблемы |
|----------|---------|----------------|
| Разные устройства должны получать обновления времени от центрального таймера без прямой привязки к конкретной реализации | **Observer** | Центральный таймер (`CentralTimeTicker`) реализует интерфейс `TimeSubject`, а устройства реализуют `TimeObserver`. Таймер уведомляет всех подписчиков при каждом тике. |
| Необходимость унифицированного управления различными устройствами (старт, стоп, отображение) | **Strategy / интерфейс DeviceController** | Все устройства реализуют интерфейс `DeviceController`, что позволяет управлять их поведением единообразно через контроллер. |
| Визуализация состояния устройств в UI с обновлением при изменении времени | **Observer + MVC** | Контроллер `Controller` подписывается на события от устройств и обновляет отображение их состояния в `VBox devicesBox` без прямого вмешательства в логику устройства. |

## Устройства и механизмы доставки времени

| Устройство | Механизм обновления | Особенности |
|------------|------------------|-------------|
| **Секундомер** | Получает время от центрального таймера и вычисляет прошедшее время с момента старта | Позволяет запускать, останавливать и сбрасывать отсчёт |
| **Таймер** | Получает время от центрального таймера и уменьшает оставшееся время | Уведомляет о завершении и отображает оставшиеся секунды |
| **Будильник** | Получает время от центрального таймера и срабатывает при достижении заданного времени | Можно включать/выключать будильник, отображает состояние "сработал" |
